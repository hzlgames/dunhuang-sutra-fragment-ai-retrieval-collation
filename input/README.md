# 输入图片目录

此目录用于存放待分析的敦煌经书残卷图片。

## 使用说明

1. 将您要分析的残卷图片放在此目录下
2. 支持的图片格式：PNG、JPG、JPEG
3. 建议图片分辨率：至少 1000x1000 像素以获得最佳 OCR 效果

## 示例

`
input/
 fragment_001.png
 fragment_002.jpg
 temp/           # 临时文件夹（不会被提交到 Git）
`

## 注意事项

- 本目录下的图片文件（*.png, *.jpg, *.jpeg）已在 .gitignore 中排除，不会被提交到 Git
- 如需提供示例图片，请在 README 中说明如何获取
