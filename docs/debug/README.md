# 调试与故障排查文档索引

本目录用于集中存放与“环境配置 / 接口调试 / 故障排查”相关的说明文档，避免占用主 `README.md`，也方便前端/后端开发者按需查阅。

建议你将现有的以下文件移动或重命名到本目录下，并在此文件中维护统一索引：

- `快速测试指南.md`
- `调试FastAPI接口+BatchAPI.md`
- 其他调试脚本说明（如有）

一个推荐的结构示例：

- `docs/debug/环境诊断与API配置.md`
- `docs/debug/FastAPI与单图接口调试.md`
- `docs/debug/BatchAPI 调试与常见错误.md`

主 README 中只保留：

- 如何启动服务；
- 对外暴露的 HTTP 接口；
- 与前端的交互方式。

具体调试步骤（如某个 curl 命令、某个 Python 脚本的完整输出示例等），请放在本目录各子文档中维护。
