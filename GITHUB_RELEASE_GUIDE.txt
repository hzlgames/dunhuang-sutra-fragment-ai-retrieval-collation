#  首次正式发布

本项目提供完整的敦煌经书残卷AI检索与校对解决方案。

##  主要功能

-  **桌面客户端**：Windows图形界面，支持拖拽上传、批处理
-  **AI识别**：Google Gemini多模态OCR
-  **经文检索**：CBETA + Gallica双引擎
-  **批处理模式**：多图并行，自动管理配额
-  **断点续传**：失败可恢复
-  **一键启动**：Windows批处理脚本

##  快速开始

详细安装说明请参考 [INSTALL.md](INSTALL.md)

**Windows用户快速启动：**
1. 下载Source code (zip)并解压
2. 安装Python 3.10+和依赖：\pip install -r requirements.txt\
3. 配置.env文件（参考.env.example）
4. 双击run_server.bat启动后端
5. 双击run_client.bat启动客户端

##  技术栈

- FastAPI + Uvicorn
- Google Gemini 3 Pro (Vertex AI)
- Python Tkinter
- CBETA API + Gallica MCP

##  重要提示

- 需要Google Cloud Platform账号和Vertex AI权限
- 请勿提交service-account-key.json到Git
- 建议图片分辨率至少1000x1000像素

##  文档

- [README.md](README.md) - 完整使用说明
- [INSTALL.md](INSTALL.md) - 快速安装指南
- [docs/vertex_directions.md](docs/vertex_directions.md) - Vertex AI配置
- [docs/vertex_batch.md](docs/vertex_batch.md) - Batch API说明
